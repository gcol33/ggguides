# Plot method for auto-fit legend plots

Plot method for auto-fit legend plots

## Usage

``` r
# S3 method for class 'gg_autofit_legend'
plot(x, ...)
```

## Arguments

- x:

  A gg_autofit_legend object

- ...:

  Additional arguments (ignored)

## Value

Invisibly returns the input object. Called for the side effect of
rendering the plot with auto-fitted and centered legend.
